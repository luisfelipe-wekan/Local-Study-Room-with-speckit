# Documents Folder

Place your PDF files here for study.

The Knowledge Extractor will:
1. Scan this folder for `.pdf` files
2. Extract text content from each PDF
3. Generate flashcards and quizzes using AI

## Supported Files

- **PDF files only** (`.pdf` extension)
- Text-based PDFs work best
- Images in PDFs are currently ignored

## Getting Started

1. Drop one or more PDF files into this folder
2. Start the application
3. Select "Study Mode" for flashcards or "Quiz Mode" for a quiz
